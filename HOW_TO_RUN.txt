STEP 2: INSTALL MONGODB
================================================================================

OPTION A - MongoDB Compass (Easiest):
  1. Download: https://www.mongodb.com/try/download/compass
  2. Install and open MongoDB Compass
  3. Connect to: mongodb://localhost:27017
  4. Done! The database will be created automatically.

OPTION B - MongoDB Atlas (Cloud):
  1. Sign up: https://www.mongodb.com/cloud/atlas
  2. Create a free cluster
  3. Get connection string
  4. Update MONGO_URI in .env file

================================================================================
STEP 3: START THE SERVERS
================================================================================

METHOD 1 - Use Startup Script (Easiest):
  
  Double-click: start.bat
  
  This will:
  - Start backend on port 5000
  - Start frontend on port 5173
  - Open browser automatically

METHOD 2 - Manual Start:

  Open Terminal 1 (Backend):
    cd d:\portfolio\backend
    npm run dev

  Open Terminal 2 (Frontend):
    cd d:\portfolio\frontend
    npm run dev

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Cannot connect to MongoDB"
Solution: 
  - Install MongoDB Compass
  - Or use MongoDB Atlas (cloud)
  - Check MONGO_URI in .env file

Problem: "Port 5000 already in use"
Solution:
  - Change PORT=5001 in .env
  - Or stop other app using port 5000

Problem: "Projects/Experience pages are empty"
Solution:
  - Run: node scripts/seedData.js
  - Or add data via Admin Dashboard
  - This is normal on first run

Problem: "Backend won't start"
Solution:
  - Check if .env file exists in backend folder
  - Verify MongoDB is running
  - Check terminal for error messages
